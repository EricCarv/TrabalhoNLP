 
# Atividade pratica de NLP (Natural Language Processing)

Projeto feito como atividade pratica da materia Natural Language Processing do curso tecnologo de Ciencia de Dados. 

Este progama utiliza de regressão logistica e tecnicas de TF-IDF para catergorizar e gerar wordclouds de fake news a partir do corpus do [Fake.br](https://github.com/roneysco/Fake.br-Corpus)



## Ferramentas utilizadas

- Python 3.14
- [Pandas](https://pandas.pydata.org/)
- [NLTK](https://www.nltk.org/)
- [Mathplotlib](https://matplotlib.org/)
- [Unicodedata](https://docs.python.org/3/library/unicodedata.html)
- [Operações com expressões regulares](https://docs.python.org/pt-br/3/library/re.html)
- [Pillow](https://pypi.org/project/pillow/)
- [scikit-learn](https://scikit-learn.org/stable/)
- [Mascaras para wordcloud](https://www.svgrepo.com/)
## Rodar Localmente

Clone o Projeto

```bash
  git clone https://github.com/EricCarv/TrabalhoNLP
```

Va para a pasta do projeto

```bash
  cd /pasta_do_projeto
```

Instale as bibliotecas

```bash
  pip install -r requirements.txt
```

Retire o comentario da linha 18 na primeira execução para baixar dependencias do nltk
